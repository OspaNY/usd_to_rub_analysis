# USD/RUB Time Series Forecasting

This project explores time series forecasting methods for predicting the USD/RUB exchange rate.

## Models

- ARIMA
- SARIMA
- LSTM (baseline deep learning approach)

## Tasks

- Time series preprocessing
- Model training and comparison
- Forecast evaluation

## Tools

- Python
- pandas, NumPy
- statsmodels
- PyTorch

## Notes

The goal of the project is to compare classical statistical methods with a simple neural network approach.
